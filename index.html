Void IDE

Website: https://voideditor.com/

An open-source cursor alternative. Built on VS Code, privacy-first, works with any LLM (local or cloud). To download, simply visit the official website and choose the appropriate version for your operating system. Follow the installation instructions provided on the website to set up Void IDE on your machine.

  • Main Community: https://discord.gg/voideditor
  • GitHub Issues: https://github.com/voideditor/void/issues
